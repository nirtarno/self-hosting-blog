Welcome to my new blog! This is my first post, and I'm excited to share my thoughts and experiences with you.

## Getting Started

This blog is built with Jekyll and hosted on GitHub Pages. It's simple, fast, and easy to maintain. I chose this setup because:

- **Markdown-based**: All posts are written in plain Markdown
- **Version controlled**: Everything is tracked in Git
- **Free hosting**: GitHub Pages provides free, reliable hosting
- **Full control**: I can customize every aspect of the design

## What to Expect

In future posts, I'll be writing about:

1. Technology and software development
2. Personal projects and experiments
3. Lessons learned and insights gained
4. Anything else that interests me!

## Adding Images

You can easily add images to your posts. Just place them in the `assets/images/` folder and reference them like this:

```markdown
![Alt text](/assets/images/your-image.jpg)
```

Or set a featured image in the front matter:

```yaml
image: /assets/images/your-image.jpg
```
